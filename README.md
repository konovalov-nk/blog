This is my blog, checked into version control for easy deployment and
backup.

The structure is simple:
- `posts` directory contains the blog posts that are available online.
- `drafts` directory contains the draft blog posts, e.g. ideas, notes.

The blog is currently hosted at [Prose.sh](blog-prose-sh) and
[blog.br11k.dev](blog-br11k-dev) is an alias. In future the plan is to
also cross-publish to other platforms, e.g. Medium, Substack, etc.
